# sip-plugin-mcp23017


The mcp23017 plugin is designed to enable I2C Relay boards with the MCP23017 chip.  You can setup each board as active high or active low.
It should automatically initialize the board.  When setting up my suggestion woudl be to not connect the power to the relays and confirm that 
it is not turing them all on at once at any stage.

Links:
* https://github.com/Dan-in-CA/SIP
* https://github.com/Dan-in-CA/sip_plugins
